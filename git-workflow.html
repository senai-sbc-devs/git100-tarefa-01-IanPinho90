Exercício

Essa é uma página destinada ao exercício de HTML/CSS e programação em pares.

Clone

Clona o repositório do GitHub para a máquina local.

Branch

Cria uma nova branch no repositório.

Commit

Salva as mudanças rastreadas com uma mensagem descritiva.

Merge

Junta a branch separada com a principal (main).

Pull Request

É um pedido para alterar o código original do GitHub.
